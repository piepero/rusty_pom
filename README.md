# RustyPom

## About the project

RustyPom is a simple command line pomodoro timer.

## Usage

```shell
; start new 25 min pomodoro or continue interrupted pomodoro
> pom

; force start of a new 25 min pomodoro
> pom -r
> pom --restart

; start new 55 min pomodoro (or continue last pomodoro)
> pom -d 55
> pom --duration 55
```
